タイプアノテーション(Collection Types)

Swiftは、値のコレクション(collections of values)を格納するため、
Array、Set、Dictionaryを提供します。

Array

Arrayは、同じタイプの値を順序付きリスト(an ordered list)に格納します。


Arrayの作成

// Intの格納できるArrayを作成
var someInts1 = Array<Int>()

[タイプ]のように簡略化して書くこともできます。

var someInts2 = [Int]()

リテラルでArrayを初期化することもできます。

var someInts3: [Int] = []           // 空のArray
var someInts4: [Int] = [1, 3, 5]    // 1, 3, 5 を含めるArray

Arrayの操作

ブラケット([])を使ってArrayにアクセス
Arrayは1ではなく、0から始まることに注意 

var someList = [2, 3, 5, 7, 3]

let someInt = someList[0]       
someList[2] = 4

print(someInt)
print(someList)


(実行結果)
2
[2, 3, 4, 7, 3]

appendで新しい値を追加

var someList = [Int]()
someList.append(7)
someList.append(2)
someList.append(2)
print(someList)


(実行結果)
[7, 2, 2]

appendはメソッド(method)というものですが、いろいろな機能を提供してくれます。
メソッドを利用して値の追加、削除、変更などができます。
詳しい内容はあとで説明します。


Arrayの繰り返し処理

Array内の値を、for–in ループで繰り返し処理することができます。

var someList = ["qwer", "asdf", "zxcv"]
for item in someList {
    print(item)
}
           

(実行結果)
qwer
asdf
zxcv

Set

Setは、同じタイプの値を順序付きのないコレクションに格納します。
Arrayとは異なり、Setは重複する値を格納することはできません。


Setの作成

// Intの格納できるSetを作成
var someInts1 = Set<Int>()

リテラルでSetを初期化することもできます。

var someInts2: Set<Int> = []            // 空きのSet
var someInts3: Set<Int> = [2, 5, 7]     // 2, 5, 7を含めるSet

Setの操作

insertで新しい値を追加

var someAnimals: Set = []
someAnimals.insert("cat")
someAnimals.insert("dog")
print(someAnimals)


(実行結果)
["cat", "dog"]

appendはメソッド(method)というものですが、いろいろな機能を提供してくれます。
メソッドを利用して値の追加、削除、変更などができます。
詳しい内容はあとで説明します。


Arrayの繰り返し処理

Array内の値を、for–in ループで繰り返し処理することができます。

var someList = ["qwer", "asdf", "zxcv"]
for item in someList {
    print(item)
}
           

(実行結果)
qwer
asdf
zxcv

Setの繰り返し処理

Set内の値を、for–in ループで繰り返し処理することができます。

var someNumbers: Set = [2, 5, 7]
for item in someNumbers {
    print(item)
}
            

(実行結果)
5
2
7

Setは、順序がないため結果が異なることがあります。


Dictionary

Dictionaryは、keyとvalueのペアを順序のないコレクションに格納します。
各valueは一意のkeyに関連付けられていて、そのvalueの識別子として機能します。


Dictionaryの作成

// Int型のkeyとString型のvalueを持つDinctionaryを作成
var someDinctionary = [Int: String]()

リテラルでDinctionaryを初期化することもできます。

var namesOfIntegers = [
    1 : "one",
    2 : "two",
    3 : "three"
]

Dinctionaryの操作

ブラケット([])を使ってArrayにアクセス

var namesOfIntegers = [
    1 : "one",
    2 : "two",
    3 : "three"
]
let nameOf2 = namesOfIntegers[2];
print(nameOf2)


(実行結果)
Optional("two")

Dinctionaryにアクセスしたら、「Optional」という結果になります。
Optionalについてはあとで説明します。


Dinctionaryの繰り返し処理

for–in ループで繰り返し処理することができます。

var namesOfIntegers = [
    1 : "one",
    2 : "two",
    3 : "three"
]

for name in namesOfIntegers {
    print(name)
}

(実行結果)
(key: 1, value: "one")
(key: 2, value: "two")
(key: 3, value: "three")
