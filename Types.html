型(Types)

　定数と変数は、保持するデータの種類を指定する型(タイプ)が必要です。
Swiftにはさまざまなタイプがありますが、
ここではInt、Double、String、Boolについて説明します。

整数(Integers)

　整数は、0、77、-2などの整数を格納できます。 整数型には、Int8, Int32, Int64, UInt32などのタイプがありますが、 今はIntだけ覚えましょう。 

let　sum: Int = 5  
print(sum)

浮動小数点数(Floating-Point Numbers)

　浮動小数点数は、3.14, 0.2など、小数部を含む数値です。 浮動小数点数型には、FloatとDoubleがあります。 今はDoubleだけ覚えましょう。 

let pi: Double = 3.14
print(pi)

文字列(String)

文字列は、「hello、world」や「swift」など一連の文字です。 文字列型には、Stringがあります。
文字列はダブルクォーテーションで囲みます。 

let greeting: String = "Hello!"
print(greeting)

論理型(Booleans)

論理型のBoolにはtrueまたはfalseだけが格納できます。 

let isRed: Bool = true
print(isRed)

型推論(Type Inference)

　定数や変数に値を代入する時、タイプアノテーション省略することができます。
Swiftがどんなタイプかを推論します。 

// 5を代入したため、SwiftはIntタイプだとわかります。
let　sum = 5 

// 3.14を代入したため、SwiftはDoubleタイプだとわかります。
let pi = 3.14

// "Hello!"を代入したため、SwiftはStringタイプだとわかります。
let greeting = "Hello!"

Int

Intのサイズは、プラットフォームによって異なります。（platform’s native word size）
32ビットプラットフォームでは、IntはInt32と同じサイズです。
64ビットプラットフォームでは、IntはInt64と同じサイズです。


UInt

UIntには、負の数を格納できません。UはUnsignedを意味する。
32ビットプラットフォームでは、UIntはUInt32と同じサイズです。
64ビットプラットフォームでは、UIntはUInt64と同じサイズです。


Float And Double

Floatは32ビットの浮動小数点数を表します。
Doubleは64ビットの浮動小数点数を表します。
