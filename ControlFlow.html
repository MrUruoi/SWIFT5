制御フロー(Control Flow)

　Swiftは、さまざまな制御フローステートメントを提供します。
Selection, Iteration(Looping), jumpなどの機能をを行います。

比較演算子(Comparison Operators)

Swiftの文(Statements)は、式の値をテストしてそれがtureかfalseかを確認する場合があります。
比較演算子は、trueかfalseかを示す論理型(Booleans)を返します。

 == 等しい(equal to)
 != 等しくない(not equal to)
 > より大きい(greather than)
 < より小さい(less than)
 >= より大きい、または等しい(greater than or equal to)
 <= より小さい、または等しい(less than or equal to)
1 == 1       // true
2 != 1       // true
2 > 1        // true
1 < 2        // true
1 >= 1       // true
2 <= 1       // false

論理演算子(Logical Operators)

論理演算子は、論理型(Booleans)を変更または組み合わせます。

 ! 論理 NOT(Logical NOT)
 && 論理 AND(Logical AND)
 || 論理 OR(Logical OR)
let a = true
!a              // false
let b = true
let c = false
b && c          // false
let d = true
let e = false
d || e          // true

if文(The if Statement)

if文は、ある値をテストしてtrueの場合、コードを実行します。

let a = 1
if a == 1 {
    print("aの値は1")       // 実行される
}

if文は、「else」句を含めることができます。
ifの後ろがfalseの場合、「else」句の後ろのコードを実行します。 

let b = 2
if b > 3 {
    print("bの値は3より大きい")        // 実行されない
} else {
    print("bの値は3より大きくない")     // 実行される
}

一連の条件をテストする必要がある場合、「else if」句を使いましょう。

let c = 3
if c == 1 {
    print("cの値は1")       // 実行されない
} else if c == 2 {
    print("cの値は2")       // 実行されない
} else if c == 3 {
    print("cの値は3")       // 実行
} else {
    print("??????")        // 実行されない
}

switch文(The switch Statement)

switch文は、いくつかのケースの中で値と一致するケースを選択します。
値がどの選択肢とも一致しない場合、「default」のケースが適用されます。

let grade = 2

switch grade {
case 5:
    print("Exellent")
case 4:
    print("Good")
case 3:
    print("Average")
case 2:
    print("Poor")       // 実行
default:
    print("Failing")
}

カンマを入れて、ケースに複数の記述することも可能

let grade = 3

switch grade {
case 5, 4, 3:
    print("Passing")            // 実行
case 2, 1:
    print("Failing")
default:
    print("Illegal grade")
}

範囲演算子(Range Operators)

Swiftには、値の範囲を表す演算子があります。

1...5       // 1 ~ 5を表す
1..<6       // 1 ~ 5を表す
3...        // 3 ~(から)を表す
...3        // ~ 3(まで)を表す

for-inループ(For-In Loops)

for–in ループは、繰り返し処理の一つです。

for index in 1...3 {
    print(index)
}


(実行結果)
1
2
3

inの後ろは、繰り返し範囲を表します。(1...3なので1から3まで)
forとinの間に、名前を指定します。(index)
このfor-in ループが実行されると、indexが範囲の最初の値(1)に初期化されます。そしてindexの値がプリントされます。
次に、indexが範囲から次の値(2)に初期化されます。そしてindexの値がプリントされます。
最後に、indexが範囲から次の値(3)に初期化されます。そしてindexの値がプリントされます。


次は、九九３の段の例です。

let x = 3
for number in 1...9 {
    print(x * number)
}

単純に繰り返すだけなら、forとinの間に、アンダースコアを入れます。

for _ in 1...3 {
    print("Hello, World!")      
}


(実行結果)
Hello, World!
Hello, World!
Hello, World!

whileループ(while Loops)

whileループは、条件がfalseになるまで一連のコードを実行します。

var i = 0
while i < 3 {
    print("Hello, World!")
    i += 1
}

whileの後ろに、条件を書きます。
最初の条件( 0 < 3 )はtrueなので {から}までのコードが実行されます。
次に、変数iの値は1になりました。条件( 1 < 3 )はtrueなのでコードがまた実行されます。
次に、変数iの値は2になりました。条件( 2 < 3 )はtrueなのでコードがまた実行されます。
最後に、変数iの値は3になりました。条件( 3 < 3 )はコードがまた実行されません。


repeat-whileループ(repeat-while Loops)

repeat-whileループは、一連のコードを実行した後、条件がfalseになるまで繰り返します。

var i = 0
repeat{ 
    print("Hello, World!")
    i += 1  
} while i < 3


(実行結果)
Hello, World!
Hello, World!
Hello, World!

continue文(The continue Statement)

continue文は、現在のループ停止し次のループ開始させます。

for number in 0...3 {
    if number == 2 {
        continue
    }
    print(number)
    // ここにジャンプ
}


(実行結果)
0
1
3

numberが2の場合、continue文が実行する条件を入れました。
continueにより、2はプリントされません。


break文(The break Statement)

break文は、制御フローの全体の実行を終了させます。

for number in 0...3 {
    if number == 2 {
        break
    }
    print(number)
}


(実行結果)
0
1
