定数と変数(Constants and Variables)

　プログラムは実行中にデータを一時的に保存する方法が必要です。そのデータの保存場所が定数と変数です。 定数は一度値が設定されると変更できませんが、変数何回も設定できます。

タイプアノテーション(Type Annotations)

　定数と変数は、保持するデータの種類を指定する型(タイプ)が必要です。 Swiftにはさまざまなタイプがありますが、ここではIntだけ紹介します。 Int型の定数と変数は、0、77、100などの整数を格納できます。 可能な値の範囲は限られていますが、詳しい内容は後で説明します。


宣言(Declarations)

　定数と変数は、使用する前に宣言する必要があります。
定数を宣言するには、letキーワードを使います。そして名前を指定し、次に型を指定します。 変数の宣言もほとんど同じです。ただし、キーワードletの代わりにキーワードvarを使います。 

// heightという名前のInt型の定数を宣言する
let height: Int    

// legthという名前のInt型の変数を宣言する
var length: Int

複数の定数または複数の変数をコンマで区切って1行で宣言できます。

var height: Int, width: Int

// タイプが同じの場合、タイプアノテーションは一番後ろに一回だけでいいです。
var length, width: Int

代入(Assignment)

定数と変数に、 = を使って値を与えることができます。

// scoreという名前のInt型の定数を宣言して、99を代入する
let score: Int  
score = 99 

// numberOfStudentsという名前のInt型の変数を宣言して、20を代入する
var numberOfStudents: Int
numberOfStudents = 20

初期化(Initialization)

　まだ値が割り当てられていない定数と変数は、初期化されていないと言われます。(uninitialized)
もちろん、代入を使って、定数や変数に初期値を与えることができます。
しかし、もっと簡単な方法があります。定数や変数の初期値を宣言時に代入することです。 

// scoreという名前のInt型の定数を宣言して、99を代入する
let score: Int = 99 

// numberOfStudentsという名前のInt型の変数を宣言して、20を代入する
var numberOfStudents: Int = 20

定数と変数の出力

let score: Int = 99 
print(score)

var numberOfStudents: Int = 20
print(numberOfStudents)


(実行結果)
99
20

識別子(Identifiers)

プログラムを作成するとき、変数、関数、マクロなどの名前を選択する必要があります。 これらの名前は識別子と呼びます。
識別子は、文字、数字、アンダースコアを含めることができますが、文字またはアンダースコアで始まる必要があります。
識別子の大文字と小文字を区別します。(case-sensitive)
多くの人はアンダースコアを避け、代わりに大文字を使用して識別子内の各単語を開始します。(Camel Case)


キーワード(Keywords)

let、varなどのキーワードは特別な意味を持ってい、識別子として使用できません。
