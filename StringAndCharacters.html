文字列と文字(String and Characters)

Characterタイプには、「a」、「あ」など一つの文字を格納できます。
Stringタイプには、「Hello」、「はい」など一連の文字を格納できます。
文字と文字列はダブルクォーテーションで囲みますが、
タイプアノテーションがないとデフォルトでStringタイプになります。 

let ch: Character = "a"     
let str: String = "abcd"

let someString = "s"        // Stringタイプ

文字列の初期化(Initializing an String)

文字列の初期化を初期はするには、ダブルクォーテーションで囲まれた文字列(文字列リテラル)を代入するか、
初期化シンタックスを利用します。 

let str1 = "Hello"
let str2 = String("Hello")

文字列の結合(Concatenating Strings)

「＋」、「＋＝」を使って文字列を結合することができます。

let str1 = "Hello"          
let str2 = " World!"
let str3 = str1 + str2      
print(str3)                 // Hello World!

var str4 = "qwer"
str4 += "ty"
print(str4)                 // qwerty

文字列補間(String Interpolation)

文字列補間は、文字列に定数や変数の値を含め新しい文字列を作ることです。

let number = 3
let msg = "定数numberの値は、\(number)"
print(msg)                              // 定数numberの値は、3

定数や変数を被演算子として使うこともできます。

let a = 1
let b = 2

let c = a + b
print(c)            // 3

文字列とfor-in ループ

Stringをfor–in ループで繰り返して、個々のCharacterにアクセスすることができます。

let str = "abcd"
for ch in str {
    print(ch)
}


(実行結果)
a
b
c
d

文字列のカウント

文字列が何文字か、countプロパティを使います。
プロパティについては、後で詳しく説明します。 

let str = "Hello World!"
let count = str.count
print(count)


(実行結果)
12

文字列の比較(Comparing Strings)

「=」演算子を使って文字列を比較することができます。

let str = "Hello World!"
let anotherStr = "Hello World!"

if str == anotherStr {
    print("Eqaul")
} else {
    print("Not equal")
}


(実行結果)
Eqaul

Unicode

コンピュータが文字を扱うために、それぞれの文字に割り当てられた番号を文字コードといいます。
文字コードの国際業界標準の一つがUnicodeです。
SwiftのStringとCharacterは、Unicodeに準拠しています。 

let someCharacter: Character  = "\u{0041}" 
print(someCharacter)

let anotherCharacter = "\u{3042}"
print(anotherCharacter)


(実行結果)
A
あ
