演算子(Operators)

　演算子は、値を確認(check)、変更(change)、組み合わせ(combine)に使う特別な記号またはフレーズです。 ここでは代入演算子と算術演算子について説明します。

代入演算子(Assignment Operators)

代入演算子「=」は、値を初期化または更新します。

let a = 2       // 定数aに2を代入
var b = a       // 変数bに定数aの値を代入

算術演算子(Arithmetic Operators)

算術演算子は、加算、減算、乗算、除算を行います。


2つの被演算子(operand)を必要な演算子は、2項演算子と呼ばれます。

let a = 1 + 1       // 1+1の結果を定数aに代入   
print(a)            // 2

let b = 3 - 1       // 3-1の結果を定数bに代入
print(b)            // 2

let c = 2 * 5       // 2×5の結果を定数cに代入
print(c)            // 10

let d = 6 / 2       // 6÷2の結果を定数dに代入
print(d)            // 3

let e = 7 % 2       // 7÷2の余りを定数eに代入
print(e)            // 1

定数や変数を被演算子として使うこともできます。

let a = 1
let b = 2

let c = a + b
print(c)            // 3

「％」を除く2項演算子では、整数と浮動小数点数を一緒に被演算子として使用できます。
例えば、IntとDoubleが混在している場合、結果の型はDoubleとなります。

let a = 6.2 / 2     // 定数aの型はDouble
print(a)            // 3.1

「/」演算子は、被演算子が両方とも整数の場合、小数部分を切り捨てます。

let a = 1 / 2       // 定数aの型はInt
print(a)            // 0.5ではなく0

「％」演算子、被演算子が両方とも整数でななければなりません。 どちらかの被演算子が整数でない場合、プログラムはコンパイルされません。

let a = 2 % 1.1       // エラー

単項(unary)演算子は、1つだけのオペランドを必要とします。

let a = -1      // -1を定数aに代入      
let b = +2      // +2を定数bに代入(+2は2と同じなので、わざわざつける必要はありません。)

複合代入演算子(Compound Assignment Operators)

Swiftには、代入演算子「=」と他の演算子を組み合わせる複合代入演算子があります。

var a = 1
a += 2      // a = a + 2 と同じ
var b = 1
b -= 2      // b = b - 2 と同じ
var c = 1
c *= 2      // c = c * 2 と同じ
var d = 1
d /= 2      // d = d / 2 と同じ
var e = 1
e %= 2      // e = 2 % 2 と同じ

優先順位と結合性(Precedence and Associativity)

演算子には優先順位があり、一部の演算子は他の演算子よりも優先されます。
優先順位を全部覚える必要はありませんが、二つ以上の演算子を使うときは気をつけましょう。 

let a = 2 + 1 * 5
print(a)            // 7

let b = (2 + 1) * 5
print(b)            // 15

結合性は、同じ優先順位の演算子を一緒にグループ化する方法を定義します。
左から右にグループ化されている場合、「左結合(left associative)」と呼ばれます。 2項演算子はすべて左結合です。 

i - j - k と　(i - j) - k は同じ

右から左にグループ化されている場合、「右結合(right associative)」と呼ばれます。 単項演算子は右結合です。

let a = 1 + -2
print(a)        // -1

Lvalues

代入演算子では、左オペランドとして「lvalue」が必要です。 「lvalue」は、コンピュータのメモリに格納されたオブジェクトを表します。 代入演算子は左オペランドとして「lvalue」を必要とするため、代入式の左側に他の種類の式を置くことはできません。 

12 = a      // エラー
a + b = 0   // エラー
-i = j      // エラー
